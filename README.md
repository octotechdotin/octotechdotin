An open source application built using React.js 18.

> **Warning**

## Running Locally

1. Install dependencies using npm:

```sh
npm install
```

2. Start the development server:

```sh
npm run dev
```
>**Development Server Running : <a href='http://localhost:5173/'>http://localhost:5173</a>**
